Our Gardens

Project Description:

The project is a website for a service called "OurGardens" under the main title "For Future." It provides an informational guide on how to create and maintain a garden, outlining a seven-step process and promoting a call to action to "Save Plants, Make Gardens."

Problem it Solves:

The project aims to solve the problem of a lack of accessible and simple guidance for individuals who want to start gardening. By providing a clear, step-by-step guide, it helps users overcome the initial hurdles of starting a garden.

Technologies Used:

The website is hosted on GitHub Pages. Projects of this type are typically built using standard web development technologies such as HTML, CSS, and JavaScript..
